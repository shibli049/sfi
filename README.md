# Svenska Inlärning (Swedish Learning)

> **⚠️ IMPORTANT**
> **Disclaimer:** Most of the content in this repository, including code and data, has been AI-generated.

Interactive tools and exercises designed to help users learn Swedish vocabulary and concepts through visual and interactive experiences.

## Tech Stack

This project is built with:
- **[Astro](https://astro.build/)**: Static site builder focusing on performance.
- **TypeScript**: For robust and maintainable code.
- **Vanilla CSS**: Custom design system with modern aesthetics (glassmorphism, animations).

## Getting Started

### Prerequisites
- Node.js (v18.0.0 or higher)
- npm

### Installation
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```

### Development
Start the development server:
```bash
npm run dev
```
The application will be available at `http://localhost:4321`.

## Deployment
This project is configured for automated deployment to GitHub Pages via GitHub Actions.
